== Description ==



Wooshark dropshipping allows to import and sell (AliExpress, Banggood, eBay, and tmart) products on your WooCommerce-wordpress store with just 1 click. 
With the WooShark app, you get better sales and increased overall productivity, 
while reducing the time and effort invested to transfer your products to your online store.



FEATURES

+ *Import product details from AliExpress, eBay, Banggood and tmart*
including 
+ *product images*
+ *product Title*
+ *product weight*
+ *product price*
+ *product description*
+ *product quantity*
+ *publish/draft a product*

+ **[Premium]** Edit the images (resize, crop, add text, add logo, add border, brightness, et..)
+ **[Premium]** Customizable and import short description
+ **[Premium]** customize and edit description using a special editor which is very easy to use
+ **[Premium]** customize, delete and edit variations (sku, price, sale price, quantity)
+ **[Premium]** set price formula which allow to automate price calculation, the user set a set of interval which are persisted and used to calculate the price.
+ **[Premium]** Customize and import Product Reviews.
+ **[Premium]** Customize and import product rating.
+ **[Premium]** Customize and import product specifications.
+ **[Premium]** Select the product categories you wish to import the products to.
+ **[Premium]** Synchronise stock and price single product [Aliexpress][eBay]
+ **[Premium]** Synchronise All products stock and price [Aliexpress][eBay]
+ **[Premium]** Apply unified price to all variations

== Installation ==
 
To install the plugin, please follow this instruction.
1. install the plugin from wordpress directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Start importing items and sell them in your WooCommerce store
4. [Premuim] Download the chrome extension  from <a href="https://www.wooshark.com/aliexpress-ebay-banggood-tmart"> Here</a>
